# CodeWorkZ 🚀
**Membunuh "Jira Fatigue" dengan Intent-Based UI bertenaga AI**

CodeWorkZ adalah ekosistem manajemen proyek generasi baru yang menggantikan birokrasi input manual (Jira/Trello) dengan pendekatan **Intent-Based UI**. Kami memungkinkan tim pengembang untuk melakukan standup, manajemen tugas, dan analisis risiko proyek hanya melalui perintah suara atau bahasa natural.

---

## 💡 The Problem
Tim pengembang sering terjebak dalam **"Jira Fatigue"**—kelelahan administratif di mana waktu produktif habis hanya untuk melakukan *drag-and-drop* kartu tugas, memperbarui status tiket, dan rapat sinkronisasi yang bertele-tele. Data sering tidak akurat karena input manual yang malas dilakukan oleh pengembang.

## 🚀 The Solution
CodeWorkZ menghilangkan kebutuhan klik manual. AI Gemini kami berperan sebagai **Orkestrator Proyek**:
1. **Speech-to-Task:** Mengubah ucapan lisan standup langsung menjadi objek tugas.
2. **Magic Sprint Board:** Papan Kanban yang ter-update otomatis secara *real-time*.
3. **Agile Risk Analytics:** Deteksi dini *burnout* (Fatigue) dan *velocity* tim berdasarkan sentimen suara pengembang.

---

## ✨ Fitur Utama
* **Voice-to-Intent Engine:** Integrasi Web Speech API yang didukung Gemini AI untuk dekonstruksi tugas.
* **Auto-Kanban:** Papan tugas yang terpisah secara otomatis menjadi "Active Sprint" dan "Done" berdasarkan niat pengguna.
* **Risk Health Dashboard:** Visualisasi metrik kesehatan tim (Burnout, Motivation, Velocity) menggunakan `Recharts`.
* **Multi-Workspace Collaboration:** Dukungan manajemen proyek terpisah untuk berbagai tim dalam satu platform.
* **Role-Based Access:** Pengaturan peran (Owner, Admin, Member) yang sinkron dengan alur database.

---

## 🛠️ Tech Stack
* **Frontend:** React, TypeScript, Tailwind CSS, Recharts (Visualisasi), Lucide-React (Icons).
* **Backend:** Node.js, Express, MongoDB (Mongoose).
* **AI Engine:** Google Gemini API 1.5 Flash.
* **Real-time:** Socket.io.
* **Utility:** Git Integration (`simple-git`).
